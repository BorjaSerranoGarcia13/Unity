Los controles son las flechas del teclado y barra espaciadora para saltar. Tecla P
para abrir el menú.

Se maneja a un personaje que tiene que acabar cada nivel sin agotar sus vidas. 
Hay que esquivar enemigos, tener cuidado de las distintas trampas y resolver puzzles
para acabar el nivel.

Modo alternativo de generación procedural de niveles infinitos. Hay una bola gigante que persigue
al jugador y destruye la parte del nivel por la que pasa. Se genera el escenario, los enemigos,
y las trampas de forma procedural.
